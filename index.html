LENGUAJE TECNICO

3DNow: Especificaciones multimedia para los procesadores de AMD. Multimedia specifications for AMD processors.
Acceso Directo: Icono especial que representa a una carpeta o un archivo y que abre o ejecuta ésta de una forma cómoda y rápida. Special icon that represents a folder or a file and that opens or executes it in a comfortable and fast way.
Acceso telefónico a redes: Aplicación que permite utilizar una línea telefónica y un módem para conectarse a otro equipo. Application that allows you to use a telephone line and a modem to connect to another computer.
Active Desktop: Escritorio activo de Windows 98 que permite tener contenido activo en el Escritorio y poner como fondo o papel tapiz una página Web, un vídeo, un sonido, etc. Windows 98 active desktop that allows you to have active content on the Desktop and set a Web page, a video, a sound, etc. as the background or wallpaper.
Active X: Tecnología de Microsoft que proporciona contenido multimedia (interactivo) a las páginas Web. Los controles ActiveX son pequeñas aplicaciones que permiten a los sitios web proporcionar contenido, como vídeos y juegos. También te permiten interactuar con contenido como barras de herramientas y tableros de cotizaciones mientras navegas por Internet. Microsoft technology that provides multimedia (interactive) content to Web pages. ActiveX controls are small applications that allow websites to provide content, such as videos and games. They also allow you to interact with content such as toolbars and quote boards while you browse the Internet.
Actualizar: Volver a cargar o mostrar el contenido de una página Web o una ventana. Reload or display the content of a Web page or window.
ADC: Circuito para convertir la señal analógica en digital, es decir, para digitalizar. Circuit to convert the analog signal into digital, that is, to digitize.
Add-ons: Añadidos o módulos que se incorporan a los que ya se tienen y que proporcionan nuevas funcionalidades. Additions or modules that are incorporated into those that already exist and that provide new functionalities.
ADSL (Asymmetric Digital Subscriber Line - Línea Digital Asimétrica de Abonado): Sistema asimétrico de transmisión de datos sobre líneas telefónicas convencionales. Asymmetric data transmission system over conventional telephone lines.
AGP: Puerto acelerador gráfico. Bus que mejora la reproducción de gráficos en el ordenador. Graphic accelerator port. Bus that improves graphics reproduction on the computer.
AIX: Versión del sistema operativo UNIX diseñada por IBM para estaciones de trabajo y grandes sistemas. Version of the UNIX operating system designed by IBM for workstations and large systems.
Alfanumérico: Cualquier combinación de números, letras y símbolos. Any combination of numbers, letters and symbols.
AMD: Empresa que fabrica los procesadores K6 o K6-2, competencia de Pentium y Pentium II. Company that manufactures the K6 or K6-2 processors, competition with Pentium and Pentium II.
Ancho de banda: Máxima cantidad de información simultánea que se puede transferir por una línea telefónica. Maximum amount of simultaneous information that can be transferred over a telephone line.
API (Aplication Program Interface - Interfaz de Aplicación del Programa): Conjunto de rutinas del sistema que se pueden usar en un programa para la gestión de entrada-salida de ficheros, etc. Set of system routines that can be used in a program for file input-output management, etc.
Aplicación: Programa que realiza una serie de funciones y con el cual trabajamos en el ordenador. Program that performs a series of functions and with which we work on the computer.
Applets: Programas desarrollados con Java para mejorar la presentación de las páginas Web que realizan animaciones, juegos e interacción con el usuario Programs developed with Java to improve the presentation of Web pages that perform animations, games and interaction with the user
Lenguaje: conjunto de definiciones (sintácticas y semánticas) utilizadas para hacer un texto que la computadora sea capaz de entender, es decir, un programa de computadora set of definitions (syntactic and semantic) used to make a text that the computer is capable of understanding, that is, a computer program.
Código: conjunto de palabras o símbolos que contienen instrucciones para la computadora. set of words or symbols that contain instructions for the computer.
Código fuente: código de un programa. La computadora lo usa para generar una versión del programa que es capaz de procesar. code of a program. The computer uses it to generate a version of the program that it is capable of processing.
Código Máquina: código que la máquina puede entender y ejecutar. code that the machine can understand and execute.
Bug: problema en el código que hace que no ejecute correctamente su función. problem in the code that causes it to not execute its function correctly.
Refactorización: proceso de cambiar un programa para mejorar su estructura interna sin cambiar su funcionamiento. Estas mejoras pueden ser para el rendimiento o para facilitar futuros trabajos en el código, por ejemplo. process of changing a program to improve its internal structure without changing its operation. These improvements may be for performance or to facilitate future work on the code, for example.
Frameworks: son como colecciones de herramientas, componentes y soluciones que puedes encontrar en un mismo paquete (el framework) que facilitan la creación de aplicaciones complejas. They are like collections of tools, components and solutions that you can find in the same package (the framework) that make it easier to create complex applications.
Queries: solicitud de información y/o datos para una base de datos. request for information and/or data for a database.
Front-End: es la parte de una aplicación web que la persona que la usa, ve e interactúa. Es ejecutada por el browser. It is the part of a web application that the person who uses it, sees and interacts with. It is executed by the browser.
Back-End: es lo que el usuario no ve en un software, como servidores y base de datos. Es un procesamiento “detrás de escena”. It is what the user does not see in software, such as servers and databases. It is “behind the scenes” processing.
Full-stack: union de Front-End y Back-End. union of Front-End and Back-End.
